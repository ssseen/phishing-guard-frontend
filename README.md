# 안전 문자 알리미

**피싱·스미싱 의심 문자를 분석해 위험 여부를 안내하는 웹 서비스**

본 서비스는 최근 증가하는 문자 사기 피해를 예방하기 위해  
사용자가 받은 문자를 입력하면 **AI 기반 분석 결과**를 시각적으로 제공하는 참고용 도구입니다.  
최종 판단과 책임은 사용자에게 있습니다.

---

##  프로젝트 개요

- **프로젝트명**: 안전 문자 알리미
- **프로젝트 유형**: 해커톤 팀 프로젝트

---

##  목적

- 세대 간 디지털 격차로 인해 문자 사기에 취약한 사용자를 돕기 위함
- 복잡한 절차 없이 문자 내용을 붙여넣기만 하면 위험 여부를 직관적으로 확인 가능

---

##  담당 역할 (Frontend)

- 서비스 전체 프론트엔드 UI/UX 구현
- 페이지 구성 및 화면 흐름 설계
- 사용자 입력 → 결과 출력까지의 인터랙션 구현
- 백엔드 API 연동 및 결과에 따른 화면 동적 처리

---

##  주요 기능

### 1. 메인 페이지
- 서비스 소개 슬라이더 (자동 슬라이드 + 수동 이동)
- 서비스 목적 및 주의 문구 안내
- 분석 페이지로 이동하는 CTA 버튼 제공

### 2. 문자 입력 안내 페이지
- 문자 복사/붙여넣기 방법을 단계별 이미지로 안내
- 사기 문자 링크 클릭 주의 메시지 제공

### 3. 문자 위험도 분석 페이지
- 문자 내용 입력 textarea
- 분석 중 로딩 애니메이션 표시
- 분석 결과에 따라 **안전 / 주의 / 피싱** 상태 시각화
- 위험 요소 및 대응 방법 리스트 출력
- 다시하기 버튼을 통한 초기화 기능

---

##  사용 기술 스택

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript

### 기타
- Google Fonts
- Custom Font (Paperlogy)
- Fetch API를 이용한 백엔드 서버 통신

---

##  API 연동

- 문자 분석을 위해 외부 AI 분석 서버와 통신
- POST 요청으로 문자 데이터를 전송
- 응답 결과에 따라 UI 색상 및 내용 동적 변경

---

## ⚠️ 유의사항

- 본 서비스는 참고용 도구이며, 실제 사기 여부에 대한 최종 판단은 사용자에게 있음
- 테스트용 문자 또는 실제 수신 문자를 그대로 입력해도 무방
